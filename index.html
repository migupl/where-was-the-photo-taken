Another way to view photos on a map

Drop photos with location metadata and they will be automatically displayed on the map as a marker.

Each marker allows you to add a title and a short description by clicking on it.

All this information can be saved in a GeoJSON file for later use. Give it a title in the field at the top of the map and click the save button. Next time drop this file with the photos to continue working on them.

Manually add a marker to the map

Add a marker is done clicking with the right button in any point of the map. This action opens a popup with a button for confirmation. This marker will be draggable.

Remove a marker from the map

Any of the markers on the map can be deleted by double clicking on it and confirming the deletion.
